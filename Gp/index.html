Recommendation Engine

Generating Meme/User Embedding
  • Used Google News 300 Word2vec pretrained model to create word2vec higher dimensional representation
  • Used BERT on meme context and user query to generate BERT based dense embedding
  • Used soft cosine similarity to get similarity score for each meme
  • * Takes into account the semantic similarity between the dimensions of the vectors
  • * Can compare vectors of different types

Generating User Queries
  • User query initially generated from chosen tags
  • User query updated by using the liked_memes/share_memes interactions from user
  • Generate feed on updated user query

Generating Feed and Ranking Memes
  • Rank memes based on Similarity Score, Popularity (likes + comments)
  • Add Random memes to avoid constricted feed
